De Escape Game is een teamspel. Samen lossen jullie de opdrachten en puzzels op.  Let op: Hoe sneller jullie het spel hebben uitgespeeld, hoe hoger jullie eindigen in het klassement. 

 

Er zijn 6 spellen die in chronologische volgorde worden gespeeld. De spellen hebben elk een eigen plek in deze ruimte. De tijdmachine vertelt jullie steeds waar jullie naar toe moeten. 

 

Elk spel levert een geheime code op. Die geheime code voeren jullie in op het scherm in de tijdmachine. Daarnaast krijgen jullie na elk spel een tandwiel. Wat jullie daar mee moeten doen zal later blijken!
